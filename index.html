Restricted Boltzmann Machines

A Probabilistic Model

Instead I’ve been using a mathematical model of this demon, which is called the Restricted Boltzmann Machine which has been largely developed by Geoffrey Hinton.

The idea behind this model is to have two layers, one which represents the neurons on the left and green, where each neuron can be either on or off, and one on the right that represents hidden units, which are equivalent to the arms of the demon, and will provide a representation of the brain activity.

Mathematically, the model is described by a joint probability distribution over the neuron and hidden unit configurations. It's a boltman distribution defined from an energy function with 3 terms.

The first term describes the tendency of each neuron to be ON or OFF.

The second term describes how each hidden unit can be more or less active, but with continuous values.

And the last term describes the interactions between the neurons and the hidden units, mediated by weights that connect directly the neurons and the hidden units. So each weight in a continuous value, which can be positive of negative

So the entire model is parametrized by those 3 parameters g and U which I won’t discuss much here, and the weights w_i_mu which we’ll discuss in length throughout this presentation.

Jérôme Tubiana

Amongst many other things, they have shown that the RBM provides a representation of brain activity as a composition of cell assemblies. I'm going to unpack this statement because it's an extremely important one for this presentation.

The idea is that RBMs capture the co-activation motifs of neurons by grouping them to the same hidden units. So if we imagine that those five neurons have a tendency of always being active together, then they will all be grouped to this blue hidden unit.

And it's the same for other groups of neurons that also tend to co-activate, and each of them will be linked to a hidden unit.

And so this creates groups of neurons, throughout the brains, that are each linked to a different hidden unit.

An important result from Thijs and Jerome is that these groups describe populations of neurons which are known and look like known functional circuits. For example, this one here looks a lot like pre-motor neurons that control the movement of the tail.

An important feature of RBMs is that one neuron can be connected to more than 1 hidden unit, like this one for example.

Aligning Representation Across Individuals

Method : Latent-aligned RBMs

So the method goes like this. We start by training a first RBM on 1 fish like Thijs and Jerome have done before. We call this RBM a teacher, and we will use it to constrain a student RBMs trained on a second fish.

The first step is to initialise this student.

We know that each hidden unit maps to certain brain regions with certain weights, and so we can map this spatial distribution into the student fish, and use it to reconstruct, in the student, the weights to that particular hidden unit, which means that it now represents the same neuronal population in both fish.

And we do this for every hidden unit to reconstruct a full weight matrix. And so with this initialization method, we have built a student RBM, which should have the same representation space as the teacher.

However, at this point the student is not at all specialized to the second fish, which is its own individual and therefore needs to be trained.

Normally, we would train it by maximazing the likelihood of the neuronal data from the second fish.

But if we do this it will slowly drift away from the teacher representation which we don't want.

With Jorge we developped a method where we also constrain the student from the teacher model by generating hidden configuration from the teacher and training the student RBM to maximise the probability of those configuration.

And we mix those two training objectives into one so the student learns to reproduce both the student neuronal configurations and the teacher hidden cofigurations.

What we found is that this training paradigm works extremely well. We can train the student RBMs about 10 times faster than we would the teacher RBMs and they converge way more reliably, meaning that they almost systematically provide good solutions to the problem.

Now because this space is shared between those two individuals and because we can generate data from those models, we can start to ask interesting questions like "do both fish encode brain activity the same way". To test this we can try to translate activity from one individual to the other.

If we take a neuronal configuration from the teacher, we can get the representation in hidden space using the conditional probability P(h|v) as I showed earlier, transfer this representation to the teacher and reconstruct it as a neuronal configuration in the student using P(v|h). For comparison we’ll also reconstruct it in the teacher’s neuronal space.

By doing this translation, we can take activity that was recorded in the teacher and try to reproduce what it would have looked like if it had also been recorded in the student.

Translated Activity is Probable under the Recipient Model

We did a lot of different tests to evaluate how realistic this translated activity is, and I'll only present one of them.

The idea is to evaluate how probable is this activity in the student model.

To do this we use the probability of neuronal configuration according to the RBM P(v). However, this cannot actually be computed, but it can be estimated through what is called the free energy.

Here I'm showing you the distribution of free energy for the teacher data evaluated in the teacher model. And as you can see, most data points are very probable, which makes sense as this is how we trained the model.

And here is the free energy of configurations reconstructed from the teacher. As you can see they are slightly less probable, which makes sense as the RBM filters out some of the details of the activity.

And now if we look at the activity that we translated from the student to the teacher we can see that it is as probable as the reconstructed configurations, and this is true of course when translating the other way around from teacher to student.

This means that the representation of the brain activity is truly shared from the student to the teacher because we can translate it from one individual to the other while still maintaining a high probability in both.

Supp : RBM vs. PCA/ICA/VAEs

PCA/ICA/VAEs are excellent for dimensionality reduction or disentangling under specific constraints, but they either assume linear/independent factors (PCA/ICA) or impose a prior encoder/latent family (VAE).

PCA enforces orthogonal directions capturing variance; ICA enforces statistical independence of components. Great for denoising/low-rank structure; less natural for overlapping assemblies or correlated motifs (decorrelation/independence can merge real modes).

Directed latent-variable model with a prescribed encoder/latent prior (typically Gaussian); powerful for generation, but the latent statistics are imposed a priori, not learned by maximum entropy, and matching pairwise statistics can be difficult in practice for shallow VAEs.

Weight structure: cRBM weights are sparse with more large-magnitude links than PCA; PCA shows predominance of small weights → less crisp assemblies

Decorrelate/independence constraints can merge real modes: in synthetic proteins, PCA/ICA produce delocalized, mixed modes; RBM separates correlated motifs that co-occur but aren’t independent.

VAE models do not fit second-order statistics.

Functional connectivity & perturbations

ICA can separate sources but lacks this generative/perturbative machinery.

They’re fine for linear/independent factors, but decorrelation/independence can merge interacting motifs and dilute assemblies
